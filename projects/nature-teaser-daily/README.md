# Nature Teaser Daily

Daily science summaries from Nature publications, posted to Twitter, Xiaohongshu, and WeChat in English and Chinese.

**Status:** Active — pipeline in setup

## Key Details
- Sources: Nature Reviews & Analysis, Nature Research Articles
- Output: Bilingual posts (EN + ZH) with vocab annotations and recap questions
- Platforms: Twitter, Xiaohongshu, WeChat

## Key Dates
- Target: Automated pipeline live by April 30, 2026

## Workflow Steps (planned)
1. Crawl t-1 articles from Nature sources
2. Select top 3–5 stories
3. Summarize each story with inline bilingual vocab
4. Add societal benefit paragraph per story
5. Generate vocabulary table
6. Generate recap question from today's content (for tomorrow's post)
7. Include recap answer from yesterday's question
8. Output formatted post ready to paste
