# Nature Teaser Daily

A solo side project that curates daily science summaries from Nature publications and distributes them across social platforms.

## Platforms
- Twitter (English + Chinese)
- Xiaohongshu / 小红书 (English + Chinese)
- WeChat (English + Chinese)

All posts require **bilingual output** — English with inline Chinese vocabulary annotations.

## Content Sources
- https://www.nature.com/nature/reviews-and-analysis
- https://www.nature.com/nature/research-articles

## Post Format
Each daily post contains:
1. **Recap question** from yesterday's post (tests reader memory)
2. **3–5 numbered stories** — each with:
   - Title
   - Summary with inline Chinese vocab (e.g., sycophancy (谄媚 / 阿谀奉承))
   - Societal benefit paragraph
   - Link to original article
3. **Vocabulary table** — key terms, Chinese translation, English definition
4. **Recap answer** at the bottom

## Tone
- External: Curious, playful, accessible to general audience
- Brief yet informative
